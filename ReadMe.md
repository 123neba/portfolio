# Minimalist Software Developer Portfolio

A high-performance, responsive personal portfolio website designed for software developers and students. This project focuses on a clean "developer-first" aesthetic, utilizing a monochrome color palette, subtle shadows, and a sophisticated typography-driven layout. It serves as a centralized hub to showcase technical projects, academic background, and core programming competencies.

## ✨ Key Features

* **Modern Professional Design:** Features a minimalist UI built with the Inter font family and Lucide icons for a crisp, high-end tech feel.
* **Fully Responsive Architecture:** Optimized with CSS Flexbox and Grid to ensure a seamless experience across mobile, tablet, and desktop devices.
* **Interactive User Experience:** Includes smooth-scroll navigation, dynamic hover states on project cards, and automated copyright year updates.
* **Structured Project Showcase:** Specialized grid layout featuring custom badges for technical stacks like Java, MySQL, and Node.js.
* **Optimized Performance:** Lightweight codebase with minimal external dependencies to ensure rapid loading speeds and high SEO efficiency.

## 🛠️ Technical Stack

* **HTML5:** Semantic structure for enhanced accessibility and search engine optimization.
* **CSS3:** Utilizes a centralized variable system in :root for easy theme management and consistency.
* **JavaScript (ES6):** Custom logic for DOM manipulation, icon initialization, and smooth scrolling behavior.
* **Lucide & DevIcons:** Integrated vector icon libraries for professional branding and technology visualization.

## 📂 Project Structure

* `index.html`: The core semantic entry point containing the Hero, Projects, About, and Contact sections.
* `style.css`: The styling engine featuring a mobile-first approach and a custom split-grid desktop layout.
* `script.js`: Handles icon creation, event listeners, and dynamic UI elements.
